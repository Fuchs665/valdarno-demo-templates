# Valdarno Demo Templates

Template di landing page pronti per il porta a porta: ristorazione ed eventi/matrimoni.
Ogni cartella contiene un singolo file `index.html` autonomo (nessuna build necessaria),
pronto per essere personalizzato e caricato su Netlify.

## Struttura

- `ristorante/index.html` — demo per ristoranti/trattorie ("Osteria della Torre")
- `eventi/index.html` — demo per location matrimoni/eventi ("Villa delle Ginestre")

## Come personalizzare una demo per un nuovo cliente

1. Apri il file `index.html` della variante che vuoi usare
2. Cerca e sostituisci: nome attività (es. "Osteria della Torre"), numero WhatsApp
   (cerca "390000000000"), indirizzo/orari/telefono nella sezione finale
3. Sostituisci le immagini placeholder (da Unsplash) con foto vere della struttura
4. Carica il file su Netlify (drag & drop) per generare la demo online da mostrare

## Nota
